## Technologies

This project is created with:

- Typescript 4.0.3
- Node.js 14.12.0
- Babel 7.11.6

## Setup

```bash
git clone https://github.com/kitsune7/revenge-of-the-pancakes
npm install
npm run build
```

## Usage

```bash
cd revenge-of-the-pancakes
npm start
```
